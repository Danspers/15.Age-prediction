## 1.NN_based_on_Logistic_regression

Классификатор на 10 классов для агрегатора магазинов одежды. Каждый класс отвечает за определённую категорию товаров. Модель представляет собой  многослойную сеть из 3х полносвязных слоёв. В обучении используется кросс-валидация. В качестве тренировочной и тестовой выборки используется датасет [fashion-MNIST](https://github.com/zalandoresearch/fashion-mnist).


## 2.LeNet_with_Adam

Тестирование алгоритма Adam (от англ. adaptive moment estimation, «адаптивность на основе оценки моментов»). Он подбирает различные параметры для разных нейронов, что ускоряет обучение модели, в сравнении с градиентным спуском (SGD), имеющий риск недообучения или переобучения, если изначально задать неподходящий шаг (learning_rate).


## 3.Fruit_Classifier

Свёрточная сеть для классификации фруктов, с применением загрузчика данных `ImageDataGenerator` и аугментации в Keras.


## 4.Residual_Network_(RasNet)

Тестирование архитектуры ResNet, в качестве альтернативы многослойной НН со свёрточными слоями.


## 5.Age_predictor

Нейронная сеть для определения возроста человека на фотографии.
Применено:
- загрузчик данных `ImageDataGenerator`
- аугментации в `Keras`
- архитектура `ResNet50`

___

Готовые скрипты описывают технический процесс работы с данными: их загрузка, масштабирование, валидация, а так же создание и обучение модели. Готовый скрипт можно применять к различным наборам даных фотмата `*.npy`
